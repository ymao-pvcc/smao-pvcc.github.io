Information Security Terms
TERMDEFINITION
1.information securityprotecting data from those who do not have permission to access it
2.authenticationprocess of verifying you are able to access it
3.authorizationprocess of telling what an authenticated user is allowed to do.
4.dual authenticationusing user ID and password and a small device that generates a random sequence of digits
5.social engineeringprocess of using people to get the information you want
6.virusinfects a host computer and then spreads.
7.wormsimilar to a virus, but it can send copies of itself to other nodes on a computer network without having to be carried by an infected host file
8.Trojan horseprogram that does some legitimate computational task but also, unbeknownst to the user, contains code to perform the same kinds of malicious attacks
9.ransomwareinvades the user’s computer by an infected email attachment
10.phishingpractice used to illegally obtain sensitive information such as credit card numbers, account numbers, and passwords

11.White Hatgood guys who keeps us safe in cyberspace
12.Black Hatbad guys who tries to infect our computers for financial gains
13.encryption/decryptionencryption-defense methods that protects our info from bad guys;decryption - breaking down that defense
14.cryptographyscience of “secret writing.” encryption/decryption
15.symmetric encryption algorithmrequires the use of a secret key known to both the sender and the receiver to open files
16.Caesar ciphershifting each character in the message to another character some fixed distance farther along in the alphabet
17.block ciphergroup or block of plaintext letters gets encoded into a block of ciphertext, but not by substituting one character at a time for each letter
18.Data Encrypytion Standard (DES)an encryption algorithm developed by IBM in the 1970s for the U.S. National Bureau of Standards
19.public key systemthe encryption key for messages to go to a particular receiver is known to everyone.
20.embedded computerscomputational devices such as chips, processors, or computers that are embedded within another system

Briefly describe how a text file can be hidden in a steganogrpahic image:

A text file can be hidden in an image file by changing the least significant bit of each byte of the image file to match the binary form of the characters in the text
