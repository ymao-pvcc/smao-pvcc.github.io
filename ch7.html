Briefly describe the five layers of TCP/IIP

  1. Layer 1. Physical Layer: govern the exchange of binary digits (bits
  2. Layer 2. Data link layer: regulates two machines connected by a communication link.
  3. Layer 3. Network layer: Create a universal addressing scheme for all network nodes and Delivering messages between any two nodes.
  4. Layer 4. Transport layer:create a “program-to-program” delivery service that delivers data .
  5. Layer 5. Application layer: implementing the end-user services provided by a network
