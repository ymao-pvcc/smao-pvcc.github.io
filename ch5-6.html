Chapter 6:An Introduction to System Software and Virtual Machines

Four things the interface between the computer and the hardware does:

  1. hides the messy details from hardware
  2. translates the information to readable lanaguages
  3. allows easy user access to the resources
  4. protection of hardware, program, data

Responsibilities of systems software:

  1. protection of details of internal struction of computer architecture
  2. presents information in a way that users can understance
  3. allows users to access to resources easily
  4. provides a secure and safe envoirment to operate
