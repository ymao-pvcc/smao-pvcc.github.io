Chapter 6: Programming Languages

Terms Definition
machine language sequences of instructions that are encoded as numeric digits.
debugging locating and correcting errors in program
assembler convert mnemonic expressions into machine language instructions.
assembly language a mnemonic system for representing programs
translator translating programs in high-level languages into machine-language programs.
compiler another name for the translation program because it compiles several machine instructions
interpreter they executed the instructions as they were translated instead of recording the translated version for future use.
cross-platform software Software that are usable across different operating systems
natural languages Human languages (English, German, etc)
formal language short,conscie lagnauges such as programming languages
machine independence Idea of machine being able to complete tasks by itself
high-level programming language a programming language with strong abstraction from the details of the computer (Python, Java, etc)
object-oriented programming (OOP) software system is viewed as a collection of units, called objects.
scripting language collection of languages that is a subset of the imperative programming languages
data types
integer a whole number, positive or negative
float a number, positive or negative, containing one or more decimals.
string numbers or alphabet surrounded by either single quotation marks, or double quotation marks.
Terms Definition
Primitive data type set of basic data types from which all other data types are constructed
data structure the conceptual shape or arrangement of data
Python collection data structures
list List items are ordered, changeable, and allow duplicate values.
tuple a collection which is ordered and unchangeable. Allows duplicate members.
set a collection which is unordered, unchangeable, and unindexed. No duplicate members.
dictionary a collection which is ordered and changeable. No duplicate members.
Terms DEFINITION
function set of instructions for performing a task
local variable variables that are created in the function and only used in that function
global variable variables that are created outside of the function and used in all functions
source program program in its original form
object program translated program
